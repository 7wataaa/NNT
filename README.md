# NNT

### 新しいタブを変更して、時計とメモを表示します

 
### 追加した機能
 - 時計 ・時間表示のみです
 - メモ ・10KBまでgoogleアカウントで同期可能  ・リンクが機能します
 
 機能は追加予定

### メモ機能の簡単な使い方
1. 逆三角形をクリック
2. メモしたい文章を入力する
3. save または sync をクリック
4. メモが保存されます

### 各ボタンの説明

*reset* → 保存されている文字列を消去します。入力中と表示中の文字列も削除されるので注意してください

*copy* → 文字列をコピーします

*save* → 入力中の文字列をローカルに保存します。5MBまで保存できます。

*sync* → 2つの動作があります。1つ目は、文字をローカルに保存の後、同期するためにリモートにも保存します(10KBまで)。  2つ目は、resetボタンのあとに押すことで発動し、リモートに保存されている文字列を同期することができます。

### 注意事項  
※保存した文字列はlocalstorageまたはchrome web storage syncに保存するため、  
**パスワードなど、大事なものを絶対に保存しないでください。**   

※chrome 拡張機能として作成。 

※macbook airで開発しました
